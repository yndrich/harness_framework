# PRD: sec_extract

## 목표
SEC EDGAR 의 10-K/10-Q 재무정보를 표준화해, 4~5개년·여러 기업을 한 엑셀에서
비교할 수 있게 한다. 애매한 항목은 사람이 검토하도록 드러낸다.

## 사용자
재무 분석을 직접 하는 사람. 기업별 재무정보를 엑셀로 모아 다년 추세와
경쟁사 비교를 뜯어본다. 표준화된 데이터로 여러 기업을 나란히 보고 싶다.

## 핵심 기능
1. 티커 입력 → 표준화된 3대 재무제표(재무상태표·손익계산서·현금흐름표)를 연도별 엑셀로
2. 여러 기업을 한 워크북에서 비교 (제표 시트의 기업 블록 + Comparison 시트의 핵심지표·마진)
3. 검토 필요 항목 표면화(Review) + 모든 값의 출처 추적(Provenance)

## MVP 제외 사항
- 사업부문별 · 지역별 매출, 비-GAAP KPI (← phase `1-segments-kpi` 에서, inline-XBRL 필요)
- 웹/GUI — CLI + 엑셀 출력만
- 외부 유료 데이터 벤더 (SEC 단독)
- 실시간·장중 데이터, 추정/예측, 밸류에이션 모델

## 디자인
- 형태: 명령행 도구가 .xlsx 를 생성. 산출물(엑셀)이 곧 UX.
- 색상: 검토 플래그만 의미색으로 — AMBIGUOUS(연노랑)/RESTATED(연주황)/GAP(연빨강).
  나머지는 무채색 + 헤더 옅은 파랑. 상세 규약은 docs/UI_GUIDE.md.
